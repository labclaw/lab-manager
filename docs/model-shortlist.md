# OCR model shortlist

This document tracks the models being considered for the lab document benchmark.

Current first-pass shortlist:

- `Mistral OCR 3`
- `GLM-OCR`
- `DeepSeek-OCR-2`
- `PaddleOCR-VL-1.5`
- `OpenRouter -> z-ai/glm-4.6v`
- `OpenRouter -> google/gemini-2.5-flash`
- `OpenRouter -> openai/gpt-4.1`
- `OpenRouter -> qwen/qwen3-vl-32b-instruct`
- `Apple Vision OCR`
- `Tesseract` baseline

Selection principle:

- Quality first on low-contrast scanned lab paperwork.
- Strong handling of tables, supplier forms, and handwritten receive dates.
- Repeatable evaluation on the lab's own scans before any production decision.
